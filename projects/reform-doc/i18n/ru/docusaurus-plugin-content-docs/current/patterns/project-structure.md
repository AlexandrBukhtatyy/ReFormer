# Структура проекта

## Рекомендуемая организация

```
your-project/
├── src/
│   └── components/                # Компоненты
│       └── Form/
│           ├── index.tsx
│           ├── form.ts           # Структура формы
│           ├── validation.ts     # Валидация
│           ├── behaviors.ts      # Поведения
│           ├── sections/          # Вложенные секции
│           │   ├── Section1.tsx
│           │   ├── Section2.tsx
│           │   └── Section3.tsx
│           └── steps/             # Шаги формы (wizard)
│               ├── Step1.tsx
│               ├── Step2.tsx
│               └── Step3.tsx
```

Такая структура держит логику формы и компоненты вместе, что упрощает поддержку и понимание кода.

## Ключевые файлы

- **form.ts** — содержит определение структуры формы (ноды, схема)
- **validation.ts** — содержит правила валидации и валидаторы
- **behaviors.ts** — содержит поведения формы (вычисляемые поля, условная логика)
- **sections/** — компоненты для вложенных секций формы
- **steps/** — компоненты для многошаговых форм (wizard)
