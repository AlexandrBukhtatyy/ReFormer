# Введение

**ReFormer** — это библиотека для работы с формами, построенная на [Preact Signals Core](https://github.com/preactjs/signals/blob/main/packages/core/README.md#guide--api). Она предоставляет декларативный способ создания сложных форм с валидацией, вычисляемыми полями и условной логикой.

## Ключевые возможности

- **Реактивное состояние** — построено на Signals для точечной реактивности
- **Типобезопасность** — полная поддержка TypeScript с выводом типов
- **Декларативная валидация** — встроенные валидаторы + поддержка кастомной валидации
- **Декларативное описание поведений формы** — вычисляемые поля, условная видимость, синхронизация полей
- **Вложенные формы** — поддержка сложных вложенных структур и динамических массивов
- **Фреймворк-агностик** — ядро работает где угодно, React-биндинги включены
- **AI-Friendly** — хорошо документированное API с понятными паттернами, легко для понимания и генерации кода ИИ-ассистентами
