# Ошибки сервера

Обработка ошибок валидации от сервера.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

- `setErrors()` — установка ошибок в поля
- Маппинг серверных ошибок на поля формы
- Отображение глобальных ошибок

## setErrors

```typescript
form.setErrors({
  email: [{ message: 'Email уже существует' }]
});
```
