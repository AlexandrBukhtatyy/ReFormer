# Условная видимость

Показ и скрытие полей с `showWhen` и `hideWhen`.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

Управление видимостью полей на основе условий:
- Показать поле "Другое" при выборе опции "Other"
- Скрыть секцию на основе выбора пользователя

## showWhen / hideWhen

```typescript
import { showWhen, hideWhen } from 'reformer/behaviors';
```
