# Условная доступность

Включение и отключение полей с `enableWhen` и `disableWhen`.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

Управление доступностью полей:
- Заблокировать поле до заполнения предыдущего
- Активировать кнопку отправки при принятии условий

## enableWhen / disableWhen

```typescript
import { enableWhen, disableWhen } from 'reformer/behaviors';
```
