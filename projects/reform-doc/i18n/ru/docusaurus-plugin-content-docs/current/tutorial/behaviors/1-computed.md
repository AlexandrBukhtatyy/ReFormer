# Вычисляемые поля

Создание вычисляемых полей с `computeFrom`.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

Вычисляемые поля автоматически рассчитывают значения на основе других полей:
- `fullName` = `firstName` + `lastName`
- `totalIncome` = `salary` + `additionalIncome`
- `age` из даты рождения

## computeFrom

import { computeFrom } from 'reformer/behaviors';
