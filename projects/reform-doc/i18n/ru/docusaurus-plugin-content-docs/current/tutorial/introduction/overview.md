# Обзор туториала

Добро пожаловать в туториал ReFormer! В этом руководстве вы пошагово создадите полноценную форму **Заявки на кредит**.

:::info В разработке
Этот раздел находится в разработке.
:::

## Что вы узнаете

По завершении туториала вы будете понимать:

- Как определять схемы форм с TypeScript
- Как создавать переиспользуемые компоненты полей
- Как добавлять валидацию (встроенную, асинхронную, кросс-полевую)
- Как реализовывать поведения (вычисляемые поля, условная логика)
- Как обрабатывать отправку формы и ошибки сервера

## Требования

- Базовые знания React и TypeScript
- Установленный ReFormer в вашем проекте

## Структура туториала

1. **Введение** — Обзор и структура формы
2. **Подготовка** — Создание FormField и компонентов полей
3. **Схема формы** — Определение интерфейса, создание схемы, декомпозиция
4. **Рендеринг** — Создание компонентов формы и шагов
5. **Поведения** — Вычисляемые поля, условная логика и другое
6. **Валидация** — Все виды валидации
7. **Поток данных** — Инициализация и сброс
8. **Отправка** — Валидация, трансформация и отправка данных

## Форма заявки на кредит

В этом туториале мы создадим многошаговую форму заявки на кредит из 6 шагов:

### Шаг 1: Основная информация

- Тип кредита (потребительский, ипотека, автокредит, бизнес, рефинансирование)
- Сумма и срок кредита
- Цель кредита
- Условные поля для ипотеки (стоимость недвижимости, первоначальный взнос)
- Условные поля для автокредита (марка, модель, год, цена)

### Шаг 2: Персональные данные

- Персональные данные (имя, фамилия, отчество, дата рождения, пол)
- Паспортные данные (серия, номер, дата выдачи, кем выдан)
- ИНН и СНИЛС

### Шаг 3: Контактная информация

- Номера телефонов (основной, дополнительный)
- Email адреса
- Адрес регистрации
- Адрес проживания (с опцией "совпадает с регистрацией")

### Шаг 4: Занятость

- Статус занятости
- Информация о компании (название, ИНН, телефон, адрес)
- Должность и стаж работы
- Доход (ежемесячный, дополнительный)
- Информация о бизнесе для ИП

### Шаг 5: Дополнительная информация

- Семейное положение и иждивенцы
- Уровень образования
- Имущество (динамический массив)
- Существующие кредиты (динамический массив)
- Созаёмщики (динамический массив)

### Шаг 6: Подтверждение

- Согласие на обработку персональных данных
- Согласие на проверку кредитной истории
- Согласие на маркетинговые рассылки
- Принятие условий
- Подтверждение достоверности данных
- Электронная подпись

### Вычисляемые поля

- Процентная ставка (на основе типа и срока кредита)
- Ежемесячный платёж
- Полное имя
- Возраст (из даты рождения)
- Общий доход
- Отношение платежа к доходу
- Общий доход созаёмщиков

Начнём!
