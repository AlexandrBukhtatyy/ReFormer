# Валидация массивов

Валидация массивов с `notEmpty` и `validateItems`.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

Валидация массивов:
- `notEmpty` — массив не пустой
- `validateItems` — валидация каждого элемента

## Использование

```typescript
import { notEmpty, validateItems } from 'reformer/validators';
```
