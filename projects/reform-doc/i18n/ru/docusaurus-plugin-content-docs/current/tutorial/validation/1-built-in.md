# Встроенные валидаторы

Все встроенные валидаторы ReFormer.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

ReFormer предоставляет следующие встроенные валидаторы:
- `required` — обязательное поле
- `min`, `max` — числовые границы
- `minLength`, `maxLength` — длина строки
- `email` — формат email
- `url` — формат URL
- `phone` — формат телефона
- `pattern` — регулярное выражение
- `number` — числовое значение
- `date` — формат даты

## Использование

```typescript
import { required, email, minLength } from 'reformer/validators';
```
