# Инициализация формы

Инициализация формы данными.

:::info В разработке
Этот раздел находится в разработке.
:::

## Обзор

- `setValue` — установка полного значения
- `patchValue` — частичное обновление
- Загрузка данных с сервера
- Предзаполнение формы

## setValue

```typescript
form.setValue(data);
```

## patchValue

```typescript
form.patchValue(partialData);
```
