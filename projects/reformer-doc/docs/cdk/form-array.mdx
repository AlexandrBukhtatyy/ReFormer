# FormArray

Headless compound-компонент для динамических массивов из `@reformer/cdk`. Массив
принадлежит модели (`model.<field>`), в схеме объявляется узлом
`{ array, item }`, а `FormArray.*` раздаёт состояние и обработчики через
render-props — разметку и стили вы пишете сами. Для готового вида используйте
`FormArraySection` из [`@reformer/ui-kit`](../ui-kit/overview).
