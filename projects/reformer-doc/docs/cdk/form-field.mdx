# FormField

Headless-анатомия поля из `@reformer/cdk`: `Root` / `Label` / `Control` / `Error` /
`Description` с автоматической провязкой `id` и `aria-*`. В отличие от готового
`FormField` из [`@reformer/ui-kit`](../ui-kit/overview), cdk-версия ничего не
рендерит сверх минимума — layout вы задаёте сами.
