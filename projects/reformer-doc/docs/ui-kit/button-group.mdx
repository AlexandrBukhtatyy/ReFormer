# ButtonGroup

Группа кнопок на shadcn: склеивает соседние элементы в единый сегмент — убирает внутренние
скругления и двойные границы. Направление задаётся через `orientation`, `ButtonGroupText`
добавляет неинтерактивную addon-подпись, а `ButtonGroupSeparator` — разделитель между сегментами.
