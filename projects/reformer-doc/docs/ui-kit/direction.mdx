# DirectionProvider

Провайдер направления письма (LTR/RTL) на shadcn / Radix. Оберните приложение или поддерево
в `DirectionProvider` один раз — вложенные Radix-примитивы (`Select`, `Tooltip`, `Slider` и др.)
подхватят направление из контекста через `useDirection` и развернут раскладку для RTL-языков.
Собственной DOM-обёртки провайдер не создаёт.
