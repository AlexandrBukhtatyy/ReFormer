# Bubble

AI-примитив пузыря сообщения чата на shadcn / Radix `Slot`. Стиль задаётся через
`variant`, сторона — через `align` (`start` — входящее, `end` — исходящее). Фон и
паддинги живут на `BubbleContent`, а `BubbleReactions` — плавающая пилюля реакций
поверх пузыря. `BubbleGroup` держит вертикальный стек сообщений.
