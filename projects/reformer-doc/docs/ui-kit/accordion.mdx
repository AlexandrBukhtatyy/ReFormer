# Accordion

Раскрывающиеся секции на shadcn / Radix `Accordion`. Compound-набор:
`Accordion` / `AccordionItem` / `AccordionTrigger` / `AccordionContent`.
Режим раскрытия задаётся через `type` (`"single"` — одна секция, `"multiple"` — несколько),
а `collapsible` позволяет свернуть открытую секцию в режиме `single`.
