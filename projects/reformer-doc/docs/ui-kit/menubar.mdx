# Menubar

Горизонтальная строка меню на shadcn / Radix `Menubar` — как строка меню десктоп-приложения
(`Файл` / `Правка` / `Вид`). Собирается из частей: `Menubar` (корневая панель), нескольких
`MenubarMenu` (по одному пункту панели), `MenubarTrigger` (заголовок-открытие), `MenubarContent`
(список в Portal) и пунктов — `MenubarItem`, `MenubarCheckboxItem`, `MenubarRadioItem`
(+ `RadioGroup`), `MenubarLabel`, `MenubarSeparator`, `MenubarShortcut`, `MenubarGroup` и вложенного
подменю `MenubarSub` / `SubTrigger` / `SubContent`. Все части несут свой `data-slot`, поэтому стиль
правится через `className` (tailwind-merge).
