# Breadcrumb

Хлебные крошки на shadcn/ui. Compound-набор:
`Breadcrumb` / `BreadcrumbList` / `BreadcrumbItem` / `BreadcrumbLink` /
`BreadcrumbPage` / `BreadcrumbSeparator` / `BreadcrumbEllipsis`.

Предыдущие уровни пути — кликабельные `BreadcrumbLink`, текущая (неактивная)
страница — `BreadcrumbPage`. Разделитель по умолчанию — иконка-шеврон, но
`BreadcrumbSeparator` принимает произвольный `children`. Длинный путь можно
свернуть через `BreadcrumbEllipsis`. Для интеграции со ссылками роутера у
`BreadcrumbLink` есть проп `asChild`.
