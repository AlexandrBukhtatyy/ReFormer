# FormField

Готовый wrapper поля из `@reformer/ui-kit` поверх headless
[`@reformer/cdk/form-field`](../cdk/form-field): одним `<FormField control={…} />`
рисует четыре зоны — **label**, **input**, **hint** (`description`) и
**validation** (`error`). Схема ниже пунктиром показывает, где что рендерится:
подпись сверху, поле, светлая подсказка под полем (слева или справа), а ошибка
валидации — уже после hint. Нужна нестандартная раскладка — спускайтесь на
уровень cdk-слотов.
