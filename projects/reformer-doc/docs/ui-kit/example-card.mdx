# ExampleCard

ReFormer-специфичная playground-обёртка примера: заголовок, описание и область с
живым примером, а кнопка в шапке переключает карточку в режим «код» (переданный
текст `code`) с copy-to-clipboard. Утилита для документации и playground — **не**
для продакшена.
