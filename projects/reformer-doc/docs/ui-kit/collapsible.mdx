# Collapsible

Раскрывающийся блок на shadcn / Radix `Collapsible`. Compound-набор:
`Collapsible` / `CollapsibleTrigger` / `CollapsibleContent`. Компонент headless —
управляет только состоянием раскрытия (uncontrolled через `defaultOpen` либо
controlled через `open` + `onOpenChange`), а оформление задаёт вызывающий код.
