# InputGroup

Обёртка контрола с аддонами на shadcn. Compound-набор: `InputGroup` оборачивает
`InputGroupInput` / `InputGroupTextarea` и `InputGroupAddon` — контейнер для иконок,
кнопок (`InputGroupButton`) и статичного текста (`InputGroupText`) по краям поля.
Выравнивание аддона задаётся через `align`, а фокус-ринг и состояние ошибки применяются
ко всей группе целиком.
