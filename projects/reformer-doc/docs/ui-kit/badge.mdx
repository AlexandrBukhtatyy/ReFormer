# Badge

Метка/значок на shadcn / Radix `Slot`. Стиль задаётся через `variant`, а `asChild`
рендерит вместо `<span>` дочерний элемент (например, `<a>` или роутер-`Link`) со
стилями метки — ховер-эффекты (`[a&]`) при этом активируются на ссылке.
