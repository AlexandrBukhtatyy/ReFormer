# Message

Сообщение чата с ИИ на shadcn. Compound-набор:
`MessageGroup` / `Message` / `MessageAvatar` / `MessageContent` / `MessageHeader` / `MessageFooter`.
Сторона реплики задаётся через `align` (`"start"` — ассистент слева, `"end"` — пользователь справа),
а `MessageGroup` собирает реплики в вертикальный тред.
