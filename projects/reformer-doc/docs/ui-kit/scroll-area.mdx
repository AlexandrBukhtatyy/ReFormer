# ScrollArea

Кастомная область прокрутки на shadcn / Radix `ScrollArea`. Тонкие полосы вместо нативного
скроллбара, стилизуются токенами темы и адаптируются к тёмной теме. Compound-набор:
`ScrollArea` (обёртка с вертикальной полосой) и `ScrollBar` (доп. полоса — например,
горизонтальная через `orientation="horizontal"`). Размер области задаётся классами (`h-*`, `w-*`).
