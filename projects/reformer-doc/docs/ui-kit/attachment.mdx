# Attachment

Чип-превью прикреплённого файла на shadcn — AI-примитив для лент вложений и превью
загрузок. Компонуется из частей: `AttachmentMedia` (иконка или миниатюра),
`AttachmentContent` с `AttachmentTitle` / `AttachmentDescription`, `AttachmentActions`
с `AttachmentAction` (кнопки), `AttachmentTrigger` (делает весь чип кликабельным) и
`AttachmentGroup` (прокручиваемая лента). Стадию загрузки задаёт `state`
(`idle` / `uploading` / `processing` / `error` / `done`), форму — `size` и `orientation`.
