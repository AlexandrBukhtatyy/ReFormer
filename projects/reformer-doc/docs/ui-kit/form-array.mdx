# FormArraySection

Готовая секция динамического массива форм поверх headless `@reformer/cdk/form-array`:
заголовок, кнопка «Добавить», карточка с меткой на каждый элемент, удаление, перестановка ↑/↓
и пустое состояние — из коробки. Элемент рендерит `itemComponent` с контрактом
`ComponentType<{ control: FormProxy<T> }>` (тот же, что у шага FormWizard). Для полностью
кастомной разметки используйте headless [FormArray](/docs/cdk/form-array) напрямую.
