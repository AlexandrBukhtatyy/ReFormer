# HoverCard

Всплывающая карточка-предпросмотр на shadcn / Radix `HoverCard`. Compound-набор:
`HoverCard` / `HoverCardTrigger` / `HoverCardContent`. Карточка появляется при
наведении курсора или фокусе с клавиатуры на триггере (обычно ссылке). Задержку
показа и скрытия задают `openDelay` / `closeDelay` на `HoverCard`, а позицию —
`side` / `align` / `sideOffset` на `HoverCardContent`.
