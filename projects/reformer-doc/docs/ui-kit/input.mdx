# Input

Текстовое поле на pure shadcn `Input`. Вариант **base** — строковый (`text` / `email` / `tel` / `url`),
вариант **number** — числовой буфер, удерживающий промежуточный ввод (`1.`, `0.05`, `-`, ведущие нули).
Для форм — `InputField`, диспетчеризующий по `type` (`number` → буфер, иначе строковый).
