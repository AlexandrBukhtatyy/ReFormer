# Marker

AI-примитив на shadcn / Radix `Slot` — строка-маркер для AI-ответов: источники, шаги
рассуждения, статусы. Compound-набор: `Marker` / `MarkerIcon` / `MarkerContent`. Стиль
строки задаётся через `variant` (`default` / `separator` / `border`), а `asChild` рендерит
вместо `<div>` дочерний элемент (например, `<a>`) со стилями маркера.
