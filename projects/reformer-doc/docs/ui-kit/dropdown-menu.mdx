# DropdownMenu

Выпадающее меню на shadcn / Radix `DropdownMenu`. Собирается из частей: `DropdownMenu` (корень),
`DropdownMenuTrigger` (открытие), `DropdownMenuContent` (список в Portal) и пунктов —
`DropdownMenuItem`, `DropdownMenuCheckboxItem`, `DropdownMenuRadioItem` (+ `RadioGroup`),
`DropdownMenuLabel`, `DropdownMenuSeparator`, `DropdownMenuShortcut`, `DropdownMenuGroup` и вложенное
подменю `DropdownMenuSub` / `SubTrigger` / `SubContent`. Все части несут свой `data-slot`, поэтому
стиль правится через `className` (tailwind-merge).
