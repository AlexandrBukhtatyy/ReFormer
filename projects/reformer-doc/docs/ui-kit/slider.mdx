# Slider

Слайдер на Radix (`Track` / `Range` / `Thumb`, role=slider). Значение — скаляр `number | null`:
одно-thumb режим сводит контракт формы к числу через `sliderAdapter` (`number` → Radix `[n]`,
`onValueChange(number[])` → `arr[0]`). Диапазон и шаг задаются `min` / `max` / `step`. Раскладка
обычная — верхнюю подпись рисует `FormField` (не inline). Для форм — `SliderField`.
