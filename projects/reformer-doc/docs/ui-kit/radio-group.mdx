# RadioGroup

Группа радио-кнопок на Radix. Вариант **base** — чистый примитив (`RadioGroup` — контейнер
`role="radiogroup"`, `RadioGroupItem` — `role="radio"`) для ручной сборки. Field-версия рендерит
опции из массива `options` (`{ value, label }`); значение — строка (`option.value | null`), каждый
вариант получает per-option `data-testid`. Для форм — `RadioGroupField`.
