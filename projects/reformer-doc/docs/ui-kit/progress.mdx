# Progress

Индикатор прогресса на shadcn / Radix `Progress` (Root + Indicator). Заполнение задаётся
числом `value` (0..100); полоса несёт `role="progressbar"` и `aria-valuenow` из коробки.
Высоту, скругление и цвет можно перекрыть через `className` (мёржатся `tailwind-merge`).
