# InputOTP

Поле ввода одноразового кода (OTP) на [`input-otp`](https://input-otp.rodz.dev/). Дефолтная раскладка
разворачивает одну `InputOTPGroup` из `maxLength` слотов; кастомная (разбивка на группы через
`InputOTPSeparator`) собирается из base-примитивов `InputOTP` / `InputOTPGroup` / `InputOTPSlot` /
`InputOTPSeparator`. Значение — `string | null` (пустой ввод → `null`). Для форм — `InputOTPField`.
