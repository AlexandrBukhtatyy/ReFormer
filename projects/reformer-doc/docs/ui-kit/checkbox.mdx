# Checkbox

Чекбокс на Radix (`button` role=checkbox + `CheckIcon`). Значение — `boolean` (контракт `checked` /
`onCheckedChange`). Раскладка **inline**: подпись из `componentProps.label` рисуется справа от контрола,
а `FormField` верхнюю подпись не рендерит (маркер `reformerLayout='inline-label'`), чтобы она не задвоилась.
Для форм — `CheckboxField`.
