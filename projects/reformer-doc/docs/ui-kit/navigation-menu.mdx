# NavigationMenu

Горизонтальное навигационное меню на shadcn / Radix `NavigationMenu`. Compound-набор:
`NavigationMenu` (корень) + `NavigationMenuList` / `NavigationMenuItem` /
`NavigationMenuTrigger` / `NavigationMenuContent` / `NavigationMenuLink` /
`NavigationMenuIndicator` / `NavigationMenuViewport`.

Пункт-триггер раскрывает выпадающую панель (`NavigationMenuContent`). По умолчанию
(`viewport={true}`) панель проецируется в общий слой `NavigationMenuViewport` под меню;
при `viewport={false}` — раскрывается инлайн под своим пунктом. Стиль пункта-триггера
задаёт `navigationMenuTriggerStyle()` — его же применяют к `NavigationMenuLink`, чтобы
простая ссылка выглядела как пункт меню. Клавиатурная навигация и `aria` — из Radix
«из коробки».
