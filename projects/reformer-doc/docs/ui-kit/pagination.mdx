# Pagination

Постраничная навигация на shadcn. Набор частей: `Pagination` / `PaginationContent` /
`PaginationItem` / `PaginationLink` / `PaginationPrevious` / `PaginationNext` /
`PaginationEllipsis`. Активная страница помечается `isActive` на `PaginationLink`
(`aria-current="page"` + стиль `outline`). Навигация — через `href` (ссылки) или
`onClick` (controlled-состояние).
