# Card

Карточка на shadcn — контейнер с шапкой, контентом и подвалом. Собирается из частей:
`Card` (корень), `CardHeader` + `CardTitle` / `CardDescription` / `CardAction`, `CardContent`
и `CardFooter`. Все части — обычные `div` со своим `data-slot`, поэтому стиль легко правится
через `className` (tailwind-merge).
