# Kbd

Клавиша или сочетание клавиш на shadcn. `Kbd` рендерит семантический `<kbd>` для одной
клавиши/ярлыка, а `KbdGroup` выстраивает несколько `Kbd` в ряд с равным зазором — для
комбинаций (например, `Ctrl` + `K`).
