# Select

Выпадающий список на Radix. Вариант **base** — чистый compound для ручной сборки
(`Select` / `SelectTrigger` / `SelectContent` / `SelectItem`). Вариант **async** — высокоуровневый:
inline `options` или асинхронный источник `resource` (стратегии `static` / `preload` / `partial`)
с поиском, пагинацией и очисткой. Для форм — `SelectField` (алиас на field-версию `async`).
