# Alert

Информационный блок на shadcn. Это compound-компонент: контейнер `Alert`
(`role="alert"`) плюс `AlertTitle` и `AlertDescription`. Стиль задаётся через
`variant` (`default` / `destructive`), а SVG-иконка первым потомком включает
двухколоночную сетку с иконкой слева.
