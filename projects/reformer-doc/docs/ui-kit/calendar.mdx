# Calendar

Календарь выбора даты на [react-day-picker](https://daypicker.dev) (v10). Вариант **base** —
чистая обёртка `DayPicker` (`Calendar` / `CalendarDayButton`) с режимами `single` / `multiple` /
`range`. Для форм — `CalendarField` (алиас на field-версию single-date: значение `Date | null`,
`selected` / `onSelect` сводятся к `value` / `onChange`). Навигация настраивается через
`captionLayout`, дни соседних месяцев — через `showOutsideDays`.
