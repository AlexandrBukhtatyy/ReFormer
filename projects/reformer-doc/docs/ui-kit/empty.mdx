# Empty

Пустое состояние на shadcn — центрированный блок с пунктирной рамкой для экранов «здесь пока
ничего нет». Собирается из частей: `Empty` (корень), `EmptyHeader` + `EmptyMedia` (иконка /
иллюстрация) / `EmptyTitle` / `EmptyDescription`, и `EmptyContent` (действия). Все части — обычные
`div` со своим `data-slot`, поэтому стиль легко правится через `className` (tailwind-merge).
