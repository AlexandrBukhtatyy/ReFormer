# Button

Кнопка на shadcn / Radix `Slot`. Стиль задаётся через `variant`, размер — через `size`,
а `asChild` рендерит вместо `<button>` дочерний элемент (например, `<a>` или роутер-`Link`)
со стилями кнопки.
