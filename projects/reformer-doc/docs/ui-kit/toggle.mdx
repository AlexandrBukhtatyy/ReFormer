# Toggle

Двухпозиционная кнопка на Radix (`button` + `aria-pressed`). Значение — `boolean` (контракт `pressed` /
`onPressedChange`). В отличие от `Checkbox` / `Switch` раскладка **НЕ inline**: подпись поля рисует
`FormField` сверху из `componentProps.label`, а контент самой кнопки (иконка или текст) задаётся через
`children`. Стиль задаётся через `variant` (`default` / `outline`), размер — через `size`.
Для форм — `ToggleField`.
