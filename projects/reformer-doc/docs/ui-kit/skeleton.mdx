# Skeleton

Плейсхолдер загрузки на shadcn — `<div>` с pulse-анимацией. Форма и размер задаются
классами (`h-*`, `w-*`, `rounded-*`) через `className`; базовые стили
(`animate-pulse rounded-md bg-accent`) можно перекрыть через `tailwind-merge`.
