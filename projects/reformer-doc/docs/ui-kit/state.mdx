# State

ReFormer-специфичные заглушки состояний экрана формы: `LoadingState` —
центрированный спиннер с подписью, пока грузятся данные; `ErrorState` — карточка
ошибки с опциональной кнопкой «Повторить». Оба несут ARIA для скринридеров:
`LoadingState` — `role="status"` + `aria-live="polite"`, `ErrorState` —
`role="alert"` + `aria-live="assertive"`.
