# Tooltip

Всплывающая подсказка на shadcn / Radix `Tooltip`. Compound: `TooltipProvider` (общий контекст
задержки, оборачивает приложение один раз) + `Tooltip` + `TooltipTrigger` + `TooltipContent`.
Подсказка появляется при наведении курсора или фокусе с клавиатуры; `Content` рендерится в
портале со стрелкой, сторона задаётся через `side`.
