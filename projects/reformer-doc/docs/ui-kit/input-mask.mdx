# InputMask

Текстовое поле с маской-подсказкой поверх pure shadcn `Input`. Шаблон `mask` (символ `9` — цифра,
прочие символы — литералы: `+`, `-`, `(`, `)`, пробел, точка) показывается в `placeholder` и помечает
формат; автоматической вставки литералов **нет**. Значение — `string | null` (пустой ввод → `null`).
Для форм — `InputMaskField`.
