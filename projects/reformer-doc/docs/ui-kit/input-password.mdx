# InputPassword

Поле ввода пароля с переключателем видимости (иконка eye/eye-off). Value-based:
значение — `string | null` (пустой ввод → `null`). Кнопка переключения появляется, когда
`showToggle` включён (по умолчанию) и значение непустое. `InputPasswordField` — form-версия
для схемы ReFormer.
