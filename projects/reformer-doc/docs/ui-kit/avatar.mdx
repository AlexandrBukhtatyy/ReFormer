# Avatar

Аватар на shadcn / Radix `Avatar`. `AvatarImage` показывает изображение, а `AvatarFallback`
подменяет его инициалами, пока картинка грузится или если ссылка недоступна. Размер задаётся
через `size` (`sm` / `default` / `lg`), а презентационные части `AvatarBadge` / `AvatarGroup` /
`AvatarGroupCount` собирают статус-индикаторы и группы аватаров.
