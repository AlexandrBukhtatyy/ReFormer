# Popover

Всплывающая панель на shadcn / Radix `Popover`. `PopoverTrigger` открывает `PopoverContent`
(рендерится в `Portal`), позиционируемый относительно триггера или отдельного `PopoverAnchor`
через `align` / `side` / `sideOffset`. Содержимое собирается из презентационных частей
`PopoverHeader` / `PopoverTitle` / `PopoverDescription` — все части несут свой `data-slot`,
поэтому стиль правится через `className` (tailwind-merge).
