# Sheet

Боковая панель на shadcn / Radix Dialog. Это compound-компонент: корень `Sheet`
(управляет состоянием), `SheetTrigger` (открывает), `SheetContent` (тело в `Portal`
с оверлеем и крестиком закрытия), `SheetHeader` + `SheetTitle` / `SheetDescription`,
`SheetFooter` и `SheetClose`. Направление выезда задаёт prop `side`
(`top` / `right` / `bottom` / `left`). Focus-trap, закрытие по `Esc`,
`aria-labelledby` / `aria-describedby` — из Radix «из коробки».
