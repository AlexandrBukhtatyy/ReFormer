# Box

DSL-контейнер: простой `<div>`-обёртка для группировки элементов в `RenderSchema`.
Layout полностью задаётся через `className` (atomic CSS / Tailwind — `flex`, `grid`, `gap-*`),
а дочерние ноды перечисляются в `children[]` схемы рендера. Box — не form-control:
у него нет `value`/`onChange`, единственный сериализуемый проп — `className`.
