# Switch

Переключатель on/off на Radix (`button` с `role="switch"`). Значение — `boolean`.
Для форм — **SwitchField** с inline-раскладкой: подпись рисуется справа из `componentProps.label`,
а верхнюю метку `FormField` не дублирует (`reformerLayout = 'inline-label'`).
