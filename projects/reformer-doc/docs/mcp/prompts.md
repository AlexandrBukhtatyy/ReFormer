# Промпты (Prompts)

В текущей версии MCP-сервер не предоставляет публичных промптов.

## Почему нет промптов?

MCP-сервер ReFormer фокусируется на предоставлении **ресурсов** (документации) и **инструментов** (report_issue). AI использует документацию из ресурсов для ответов на вопросы и помощи с кодом.

Промпты доступны только в режиме отладки для разработки MCP-сервера.

## Как получить помощь?

Просто задайте вопрос AI в вашей IDE:

```
Создай форму регистрации с email, паролем и подтверждением пароля
```

```
Как добавить валидацию для поля телефона?
```

```
Почему моя форма пересоздаётся при каждом рендере?
```

AI получит актуальную документацию через ресурсы MCP-сервера и поможет с решением задачи.
