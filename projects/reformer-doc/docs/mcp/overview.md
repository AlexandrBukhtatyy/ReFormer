# Обзор

**@reformer/mcp** — это MCP-сервер (Model Context Protocol), который предоставляет AI-ассистентам полный контекст библиотеки ReFormer для помощи в разработке форм.

## Что такое MCP?

[Model Context Protocol](https://modelcontextprotocol.io/) — это открытый стандарт от Anthropic для интеграции AI-ассистентов с внешними инструментами и данными. MCP-серверы предоставляют:

- **Resources** — доступ к документации и данным
- **Tools** — инструменты, которые AI может вызывать

## Возможности @reformer/mcp

### Документация

AI получает доступ к актуальной документации ReFormer:

| Ресурс | Описание |
|--------|----------|
| `reformer://docs` | Полная документация |
| `reformer://api` | Справочник API |
| `reformer://examples` | Примеры кода |
| `reformer://troubleshooting` | Решение типичных проблем |

### Сбор обратной связи

Инструмент `report_issue` позволяет AI автоматически сообщать об ошибках и найденных решениях. Это помогает улучшать документацию и библиотеку.

### Помощь в разработке

С MCP-сервером AI может:

- Отвечать на вопросы о ReFormer на основе актуальной документации
- Помогать с созданием форм, валидации и поведений
- Объяснять ошибки и предлагать решения
- Показывать примеры паттернов использования
