# Обзор

**@reformer/mcp** — это MCP-сервер (Model Context Protocol), который предоставляет AI-ассистентам полный контекст библиотеки ReFormer для помощи в разработке форм.

## Что такое MCP?

[Model Context Protocol](https://modelcontextprotocol.io/) — это открытый стандарт от Anthropic для интеграции AI-ассистентов с внешними инструментами и данными. MCP-серверы предоставляют:

- **Tools** — инструменты, которые AI может вызывать
- **Prompts** — предустановленные шаблоны для типовых задач
- **Resources** — доступ к документации и данным

## Возможности @reformer/mcp

### Генерация кода

- Создание форм по текстовому описанию
- Генерация TypeScript типов
- Создание схем валидации
- Настройка поведений (computed fields, conditional fields)

### Проверка качества

- Анализ кода на типичные ошибки
- Проверка правильности импортов
- Рекомендации по структуре проекта

### Помощь в разработке

- Объяснение ошибок с примерами исправлений
- Примеры паттернов использования
- Актуальная документация API
